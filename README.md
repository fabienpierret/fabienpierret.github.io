# Portfolio GitHub Pages

Ce portfolio sera hébergé sur `fabienpierret.github.io`.

## Instructions de Déploiement

1. **Créer le repo GitHub** :
   - Aller sur github.com
   - Créer un nouveau repo nommé : `fabienpierret.github.io`
   - Le rendre public

2. **Uploader les fichiers** :
   - Uploader le fichier `index.html` dans le repo
   - Activer GitHub Pages dans les Settings du repo

3. **Accéder au portfolio** :
   - Le site sera accessible à : https://fabienpierret.github.io

## Structure

- `index.html` : Page principale du portfolio
- Design moderne et responsive
- Présente les compétences et projets

## Personnalisation

Le fichier `index.html` peut être modifié pour :
- Ajouter plus de projets
- Modifier les couleurs
- Ajouter des sections
